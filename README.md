# HeartCare

HeartCare 是一个基于 iOS 的健康数据可视化应用，用于查看心率、HRV、呼吸频率与睡眠阶段，并提供多维度背景叠加与数据同步能力。

## 主要功能

- 心率、HRV、呼吸频率趋势图展示
- 睡眠背景叠加（统一睡眠/分阶段睡眠）
- 呼吸图睡眠阶段筛选（深度、核心、安宁、清醒）
- Core Motion 活动背景叠加（步行、跑步、静止）
- 自动增量同步（每 10 分钟）与手动同步（近 7 天）
- 首次安装支持历史数据回填

## 技术栈

- Swift / SwiftUI
- Apple Charts
- HealthKit
- Core Motion
- SQLite（本地持久化）

## 数据来源与说明

- 心率、HRV、呼吸、睡眠数据来自 HealthKit
- 活动状态（步行/跑步/静止）来自 Core Motion
- 睡眠阶段显示基于原始睡眠区间数据进行渲染（并做最小去重）

## 运行环境

- Xcode 15+
- iOS 17+
- 需在真机授权 HealthKit 与 Motion 权限

## 快速开始

1. 克隆仓库
2. 使用 Xcode 打开 `HeartCare.xcodeproj`
3. 选择签名与目标设备
4. 运行 App 并授权健康与运动数据访问
